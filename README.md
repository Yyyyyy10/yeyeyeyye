* 远程仓库配置
				   - 注册账户：
	   -  类似普通的网站新用户注册，使用邮箱注册即可。
   - 每个文件的修改，删除，Git都能跟踪
   - 新建仓库：
   -  点击右上角，加号，new repository
* 创建与合并分支
   - 在版本回填退里，你已经知道，每次提交，Git都把它们串成一条时间线，这条时间线就是一个分支
   - HEAD严格来说不是指向提交，而是指向master，master才是指向提交的，所以，HEAD指向的就是当前分支
   - 分支管理策略
   - 创建一个dev分支。
   - 修改readme.txt内容、
   - 添加到暂存区。
   - 切换回主分支(master)
   - 合并dev分支，使用命令 git merge –-no-ff -m “注释” dev
